Taste Tour

The site uses a MySQL database with places stored in it. When the user searches for a place it uses their geolocation and plots out a path to the five closest restaurants matching their query. Another teammate and myself worked on the map routing feature from front-end to back-end.

Technology Used:

  • Vue.js
  • Node.js
  • Mapbox API
  • MySQL
  • Express
